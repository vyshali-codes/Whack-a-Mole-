# Whack-a-Mole Mania 🐹🔨

A fast-paced, vibrant, and satisfying Whack-a-Mole game built with React, Tailwind CSS, and Framer Motion.

## Features

- **Dynamic Gameplay**: Moles pop up with varying speeds, positions, and animations.
- **Mole Types**:
  - **Normal**: Standard mole, 1 hit = 1 point.
  - **Golden**: Rare and fast, 1 hit = 5 points!
  - **Stubborn**: Tougher mole, requires 2 hits to defeat.
- **Smooth Animations**: Powered by Framer Motion for bouncy pop-ups, snappy hiding, and satisfying whack effects.
- **Difficulty Scaling**: The game gets faster as your score increases.
- **High Score Tracking**: Compete against your best score (saved locally).
- **Pause/Resume**: Take a break without losing your progress.
- **Responsive Design**: Polished UI that looks great on different screen sizes.

## How to Play

1. Click **START GAME** to begin the 30-second countdown.
2. Click on the moles as they appear in the holes.
3. Watch out for special moles!
4. Try to beat the high score!

## Tech Stack

- **React**: Frontend framework.
- **Tailwind CSS**: Styling and layout.
- **Framer Motion**: Advanced animations and transitions.
- **Lucide React**: Beautiful icons.
- **shadcn/ui**: Accessible UI components.

## Development

To run the project locally:

```bash
npm install
npm run dev
```

Enjoy the mania!
